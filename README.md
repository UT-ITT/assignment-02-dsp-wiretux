# Karaoke: Freude Schöner Götterfunken

## How to Play
1.  Press `SPACE` to begin the song.
2.  Sing and try to hit the notes of *"Freude Schöner Götterfunken"*.
3.  A vertical slider represents your current pitch within a octave. You don't need to hit the octave. Meaning you can sing the song higher and lower pitched if you want.
4.  Note Blocks fly in from the right. Match your pitch to the block's height exactly when it reaches the slider area. The current active note will turn **Light Blue**.

## Scoring
| Accuracy | Result | Points | Block Color |
| :--- | :--- | :--- | :--- |
| **Perfect** (> 60%) | Good job! | **100** | Green |
| **Good** (10% - 60%) | Still close enough! | **50** | Yellow |
| **Miss** (< 10%) | Missed! | **0** | Red |

## Retry
If you want to retry just press `SPACE` again at the end to restart

# Whistle Input
1. Whistle **down** to press the `arrow key down`
2. Whistle **up** to press the `arrow key up`
3. You need to hit 5 descending or ascending frequencies in a row
4. Between each input there is a delay of 1 Second
